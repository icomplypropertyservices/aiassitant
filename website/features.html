Features

Built for real business operations

Not a single chatbot — a full AI operations stack: org structure, agents, knowledge, and billing you can explain to a client.

🗂️

Workspace model

Company → Projects → Tasks. Own multiple companies, allocate agents, and keep client work separated without new logins.

👑

Orchestrator hierarchy

Main AI Orchestrator stays on top. Leads manage specialists. Delegate work down the tree with clear ownership.

💬

AI chat that bills fairly

REST chat that works on Vercel. Choose fast or quality models. Token usage streams into a live meter and wallet.

📖

Training & files

Knowledge folders, uploads, and agent programs. Grant agents only the files they need. Ready for GCS / Dropbox backends.

🔌

Integrations vault

Connect apps with OAuth or API keys. Secrets encrypted at rest. Probe connections and attach them to specific agents.

📱

Web + mobile

Capacitor shells for iOS and Android share the same account. Payments route to the secure web app when store rules require it.
